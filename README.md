# 📧 AI Spam Email Detection System

This project is a Machine Learning based Spam Email Detection System built using:

- Python
- Scikit-learn
- TF-IDF
- Multinomial Naive Bayes
- Streamlit

## Features
- Detects Spam and Ham Emails
- Interactive Web UI
- Real-time prediction
